# rust-tennis-game
CUI Tennis Game by Rust

## 参考: ゲームを作りながら Rust 言語の基本を解説
- [前編](https://www.youtube.com/watch?v=LW9hT0nY51Y)
- [後編](https://www.youtube.com/watch?v=a01nZTFVj6Q)

## Rustのインストール
- `curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`
  - `rustup update`
  - `rustup self uninstall`
- `rustc --version`

## プロジェクトのセットアップ
- `cargo init tennis`
- `cd tennis`
- `code .`
- `cargo run`

- `cargo run`
